# Markdown Slider

プレゼンテーション用Plackアプリ、まだ構想段階...

プレゼンする時は１ページずつめくれれば良いけど後日落とした人はCtrl-Fで検索できたりインデックス見れると便利だよなとか思って考えてる。

- left key ... back
- right key ... go
- down key ... index
    - enter key ... go to selected page
    - delete key ... cancel
- shift + left ... go to first page
- shift + right ... go to last page
