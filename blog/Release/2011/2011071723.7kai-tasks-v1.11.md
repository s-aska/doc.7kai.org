# 7kai Tasks v1.11 release

<http://tasks.7kai.org/>

4日しか空いていませんが 7kai Tasks v1.11 出しました。

長い長いUI調整の度はまだまだ続くのでした。

# 修正箇所
- CSS調整
- コメント入力を「コメント」をクリックから起動
- 通知の実装（ 画面右上 ）
- Chrome 拡張機能をインストールしていない場合に催促のメッセージ表示
- コメントの自動リンク
- コメントに長いURLを入力した場合に適切に折り返す
- 依頼者の変更に対応
- ホバーで出現するメニューの開閉処理を調整、500msのdelayを設定
- 期限をクリックした際にタスク編集画面表示
