## ショートカット
- command + [ ... インデントダウン
- command + ] ... インデントアップ
- command + / ... コメントアウト
- option + command + l ... 行番号表示
- option + command + 1 ... パラグラフ開閉

### 一応...
- command + s ... 保存
- command + w ... 閉じる

## 調査中
- 保存時に無駄なスペースを取りたい
- 選択範囲をのみ置換したい
